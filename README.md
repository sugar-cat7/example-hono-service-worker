# App

## Requirements

### Runtime

- node.js
- docker

## Local Development Server

### API

#### Running the Server

To start the server:

```bash
$ pnpm install
```

- API Server
```bash
$ cd services/api-server
$ pnpm dev
```

- Web UI
```bash
$ cd services/web
$ pnpm dev
```

#### Swagger UI

Access the Swagger UI at:

- [http://localhost:8787/swagger-ui](http://localhost:8787/swagger-ui)
